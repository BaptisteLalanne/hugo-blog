# About the website

This website uses [**Hugo**](https://gohugo.io/)


## Getting started

```bash
git clone git@github.com:BaptisteLalanne/hugo-blog.git
cd hugo-blog
hugo server --disableFastRender
```


To render the website for the production simply use:
```
hugo
```
